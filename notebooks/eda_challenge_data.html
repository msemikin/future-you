Synt Transactions 10M¶

From challenge organizers: https://docs.google.com/document/d/1QmG5K65lNzFEaMHZsU8g8yIt4ob9sKLl3bexorMvDXY/edit

  • Category = self explanatory
  • tstamp = timestamp - this is the actual time the saldo on the account was effected
  • arvopvm = value date
  • kirjauspvm = timestamp of logging payment
  • maksupvm = timestamp of payment
  • tilinro = account number
  • rahamr = amount of payment

  • saldo = balance of account

  • vientiselitekd = type of entry (examples: 701 = subscription, 721 = card payment, 760 = loan payment, 710 = deposit, 720 = withdrawal, please ask in the Brella booth if you need these in more detail)

  • taplajikd = type of payment (100-199 debited (outgoing money), 200-299 cancellation of debited sum, 500-599 refund, 600-699 cancellation of refund (incoming money))

  • bic_saaja = BIC of recipient

  • viite = reference code of payment

  • iban_saaja = IBAN of recipient
